# Dataset Paths

The tutorial plugin builds fallback URLs by appending dataset-relative paths to
the configured fallback base URL.

Examples:

- `datasets/openneuro/ds000102/dataset_description.json`
- `datasets/xnat-tutorial/tcia_collection_smallest/archive.zip`
- `datasets/xnat-tutorial/tcia_dicom_intro/archive.zip`
- `datasets/xnat-tutorial/tcia_mouse_astrocytoma_mri/archive.zip`
