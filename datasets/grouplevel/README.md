# Group-Level Tutorial Sources

This directory tracks sources used by the Group-Level tutorial dataset import
flows. The catalog is in `sources.yml`.

The small MSD Hippocampus archive is mirrored here as a raw GitHub fallback:

- `msd/Task04_Hippocampus.tar`

Other referenced sources are too large for normal GitHub raw repository files.
They are still listed in `sources.yml` with original URLs, sizes, and credit so
the fallback work is explicit. Use GitHub release assets, object storage, or an
equivalent large-file endpoint for those if they need mirrored downloads.
